# Capacitated-Facility-Location-Problem
Esse é um repositório contendo meu programa em C++ usando o Solver CPLEX para resolver o problema de localização de facilidades capacitado que fiz durante minhas aulas de Programação Linear-Inteira.

Para rodar o programa em Linux use:
```bash
make
```
Em sequência use:
```bash
./plfc TPI_F_0.txt
```
